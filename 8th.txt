Start Minikube:

```bash
minikube start --driver=docker
```

Verify cluster:

```bash
kubectl get nodes
```

Output should show:
minikube Ready

---

Create Project Folder

```bash
mkdir ai-k8s-app
cd ai-k8s-app
```

---

Create AI Application

Create file:

```bash
nano app.py
```

Paste:

```python
from flask import Flask, request, jsonify

app = Flask(__name__)

@app.route("/")
def home():
    return "AI Model Running in Kubernetes"

@app.route("/predict", methods=["POST"])
def predict():
    data = request.json["number"]
    result = data * 2
    return jsonify({"result": result})

if __name__ == "__main__":
    app.run(host="0.0.0.0", port=5000)
```

---

Create requirements file:

```bash
nano requirements.txt
```

Paste:

```txt
Flask
```

---

Create Dockerfile

```bash
nano Dockerfile
```

Paste:

```dockerfile
FROM python:3.9

WORKDIR /app

COPY . .

RUN pip install flask

EXPOSE 5000

CMD ["python","app.py"]
```

---

Build Docker Image

```bash
sudo docker build -t ai-model .
```

Check image:

```bash
sudo docker images
```

---

Load Image into Minikube

```bash
minikube image load ai-model
```

---

Create Kubernetes Deployment

Create file:

```bash
nano deployment.yaml
```

Paste:

```yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: ai-app

spec:
  replicas: 2

  selector:
    matchLabels:
      app: ai-app

  template:
    metadata:
      labels:
        app: ai-app

    spec:
      containers:
        - name: ai-container
          image: ai-model
          imagePullPolicy: Never
          ports:
            - containerPort: 5000
          resources:
            requests:
              cpu: "100m"
            limits:
              cpu: "500m"
```

Deploy it:

```bash
kubectl apply -f deployment.yaml
```

Check pods:

```bash
kubectl get pods
```

---

Create Kubernetes Service

Create file:

```bash
nano service.yaml
```

Paste:

```yaml
apiVersion: v1
kind: Service
metadata:
  name: ai-service

spec:
  type: NodePort

  selector:
    app: ai-app

  ports:
    - port: 5000
      targetPort: 5000
      nodePort: 30007
```

Deploy service:

```bash
kubectl apply -f service.yaml
```

Check:

```bash
kubectl get svc
```

---

Access Application

```bash
minikube service ai-service
```

Browser opens showing:
AI Model Running in Kubernetes

---

Enable Metrics Server

```bash
minikube addons enable metrics-server
```

Verify:

```bash
kubectl top pods
```

Example output:
ai-app-xxxxx 30m 100Mi

---

Create Horizontal Pod Autoscaler

```bash
kubectl autoscale deployment ai-app --cpu-percent=50 --min=2 --max=10
```

Check autoscaler:

```bash
kubectl get hpa
```

Example:
ai-app cpu: 10%/50% min 2 max 10

---

Generate Load

```bash
while true; do curl $(minikube service ai-service --url); done
```
