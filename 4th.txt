#  Jenkins Automated Build using GitHub Webhook

## Aim

Configure Jenkins to automatically trigger and execute a Python program when code is pushed to a GitHub repository.

---

##  Requirements

* Jenkins installed & running
* Git installed
* GitHub account
* Python installed
* Internet connection

---

## 🧪 Steps

### 1. Start Jenkins

```bash
sudo systemctl start jenkins.service
sudo systemctl status jenkins.service
```

Open in browser:

```
http://localhost:8080
```

Login:

```
Username: admin
Password: admin123
```

---

### 2. Install Plugins

Go to:

```
Manage Jenkins → Manage Plugins → Available Plugins
```

Install:

* Git Plugin
* GitHub Plugin

Restart Jenkins.

---

### 3. Create GitHub Repository

Create a new repo and add file `add.py`:

```python
a = 10
b = 20
print("Addition is:", a+b)
```

Commit and push.

---

### 4. Create Jenkins Job

* Go to Dashboard → **New Item**
* Name: `Addition_Project`
* Select: **Freestyle Project**
* Click **OK**

---

### 5. Connect GitHub Repo

* Go to **Configure**
* Source Code Management → **Git**
* Enter:

```
https://github.com/yourusername/yourrepository.git
```

Branch:

```
*/main
```

---

### 6. Add Build Step

* Go to **Build Section**
* Add: **Execute Shell**

```bash
echo "Build Started"
python3 add.py
echo "Build Finished"
```

---

### 7. Enable Webhook Trigger

* Go to **Build Triggers**
* Select:

```
GitHub hook trigger for GITScm polling
```

---

### 8. Get System IP

```bash
ip addr
```

Example:

```
172.16.3.199
```

Webhook URL:

```
http://172.16.3.199:8080/github-webhook/
```

---

### 9. Configure GitHub Webhook

Go to:

```
Repository → Settings → Webhooks → Add Webhook
```

* Payload URL:

```
http://172.16.3.199:8080/github-webhook/
```

* Content type:

```
application/json
```

* Event:

```
Just the push event
```

---

### 10. Trigger Build

Modify `add.py`:

```python
print("Webhook Triggered Successfully")
```

Push:

```bash
git add .
git commit -m "Webhook test"
git push origin main
```

---

### 11. Verify Build

Go to:

```
Jenkins → Job → Build History → Console Output
```

Expected Output:

```
Build Started
Addition is: 30
Webhook Triggered Successfully
Build Finished
Finished: SUCCESS
```

---

## Result

Jenkins successfully triggers an automated build using GitHub Webhook on every push.
